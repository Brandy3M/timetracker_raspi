Herunterladen, Bearbeiten & Hochladen:
git clone https://github.com/username/projektname.git

cd projektname

> open file and perform changes > safe

git Status (Displays files with changes e.g., modified: README.md)

Add to staging-Area:
git add README.md

Add to repository/Project:
git commit -m "REASON: README aktualisieren"

git push origin main (Push in main-Branch)

git push origin feature/meine-aenderung (Push in ein feature-Branch)



Herunterladen: git Clone

Common history of files:
git log
